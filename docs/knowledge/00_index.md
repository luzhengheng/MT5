## 使用指南

### 知识贡献

1. **确定分类**：根据知识主题选择合适的分类目录
2. **命名规范**：使用描述性文件名，格式：`主题_日期.md`
3. **内容结构**：参考[知识文档模板](../templates/knowledge_template.md)
4. **索引更新**：新增知识后在此索引中添加条目

### 知识检索

- **按分类浏览**：直接进入对应子目录查看
- **关键词搜索**：使用git grep或文档工具搜索
- **关联导航**：通过文档内的交叉引用发现相关知识

### 质量维护

- **定期review**：每月检查知识的时效性和准确性
- **版本管理**：重要知识点记录变更历史
- **反馈机制**：知识使用者可提出改进建议
